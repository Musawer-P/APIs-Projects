Country Explorer is a simple and interactive web app that allows users to search or select any country in the world and instantly explore key information such as its flag, capital, population, area, currency, time zone, and language — all fetched in real-time from the REST Countries API.

Features:

Search by Country Name – Type the country name (like Japan or USA) and get instant results.
Dropdown Selection – Choose a country directly from a list to view its details.
Flag Display – Shows the country’s official flag.
Capital & Area – Displays the main capital city and total area (in km²).
Population – Shows the total population formatted neatly.
Currency – Displays the country’s currency name and symbol.
Timezones – Lists all time zones for that country.
Languages – Displays all official languages spoken.

Technologies Used:

HTML5 – Structure
CSS3 – Styling
JavaScript (ES6) – Logic and API integration
REST Countries API
 – Country data source
