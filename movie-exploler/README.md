Movie Explorer (Movie Nest)

Overview:

The Movie Explorer allows users to search any movie using the OMDb API and browse trending movies using the TMDb API.
It shows high-quality posters, ratings, runtime, and descriptions.

Features:

Movie Search (OMDb API)
Enter a movie name

Instantly loads:

Poster
Title
Plot
Runtime
IMDb rating
Fallback poster image if unavailable
Trending Movies (TMDb API)
Automatically loads trending movies of the day

Shows:

Poster
Movie title
Rating
Uses placeholder poster when missing

APIs Used
OMDb API
https://www.omdbapi.com/?apikey=YOUR_KEY&t=MOVIE_NAME

TMDb API
https://api.themoviedb.org/3/trending/movie/day?api_key=YOUR_KEY
