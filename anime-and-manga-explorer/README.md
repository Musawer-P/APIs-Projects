Anime & Manga Explorer

Anime & Manga Explorer is a simple and interactive web application that allows users to search for anime, view detailed information, and explore trending shows. The project uses the public Jikan API, which provides data from MyAnimeList.

This project is ideal for beginners who want to practice JavaScript, API integration, and dynamic UI rendering.

Features:

Search Anime
Users can search for any anime by typing in the search bar and pressing Enter.

The app displays:
Anime image
Title
Description
Genres
Episodes
Score
Trending Anime

The homepage includes a "Trend" section that automatically loads the top airing anime from the Jikan API. Each trend box 

displays:
Anime poster
Title
The trend section refreshes with real data every time the page loads.
Clean UI Layout

The design includes:
A trending slider-style layout
A main card section for search results
Responsive display structure

Technologies Used:
HTML5
CSS3
JavaScript (Vanilla JS)
Jikan API (v4)
