SpaceX Launch Tracker

A simple and interactive web application that tracks SpaceX launches using the official SpaceX public API. This project displays the latest launch, next upcoming launch, and a list of upcoming missions in a clean and user‑friendly interface.

Features:

Latest SpaceX launch details
Next upcoming launch information
Live countdown timer for the next launch
List of upcoming launches
Automatic status detection (success / failure / upcoming)
Clean and responsive UI
Uses real-time data from SpaceX API

Technologies Used:

HTML5
CSS3
JavaScript (Vanilla JS)
SpaceX REST API (v4)

API Used:

This project uses the official SpaceX public API:
Base URL: https://api.spacexdata.com/v4


How to Run the Project:

Download or clone the repository
Open the project folder
Open index.html in your browser
That’s it — no server or build process needed.

How It Works:

JavaScript fetches launch data from the SpaceX API
Launch status is detected automatically:
Neutral color → Upcoming / unknown
Countdown timer updates every second for the next launch
Screens & UI
Latest Launch Card
Next Launch Card with Countdown
Upcoming Launches List
The UI is designed to be simple, readable, and informative.

Known Limitations:
Launch times depend on SpaceX API accuracy
Some launches may not have full details
